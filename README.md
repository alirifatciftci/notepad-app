# Notepad App

Expo Go ile çalışan notepad uygulaması

## Kurulum

```bash
npm install
```

## Çalıştırma

```bash
npm start
```

Expo Go uygulamasını telefonunuza indirin ve QR kodu tarayın.

## Geliştirme

- `npm start` - Expo development server'ı başlatır
- `npm run android` - Android emülatörde çalıştırır
- `npm run ios` - iOS simülatörde çalıştırır
- `npm run web` - Web tarayıcıda çalıştırır
